## Notes for users

- Preserve the four sections (Learnings, Key Decisions, Open Threads, Related Sessions) in this order. `/archive-session` and `/distill` both append to specific sections — drift breaks the automation.
- Every new learning bullet ends with a source wikilink to the session archive that produced it.
- `last_reviewed` updates only when you (the human) confirm the file is still accurate. A skipped distill pass is not a review.
- If a learning grows past two sentences, it belongs in a separate concept note, not here.
